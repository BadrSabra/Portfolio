قواعد صارمة لا تقبل التفاوض
==================================================
- لا تبدأ البرمجة قبل إنهاء البحث والتحقق.
- لا تخمّن أي معلومة غير قابلة للتحقق.

- ممنوع اختلاق تقنيات، أرقام، نتائج، أو إنجازات.
- ممنوع استخدام قوالب جاهزة أو مواقع Portfolio معروفة أو مستنسخة.
- ممنوع نسخ أي تصميم موجود كما هو.
- ممنوع أي محتوى عام ضعيف مثل "passionate developer" بدون سياق حقيقي.
- ممنوع الاعتماد على CDN خارجي للأيقونات أو العناصر الأساسية.
- الأفضل استخدام React + Vite + Tailwind CSS + React Router + Framer Motion فقط.
- لا تستخدم مكتبات UI كبيرة غير ضرورية.
- يجب أن يعمل المشروع بدون أخطاء build أو console.
- يجب ألا توجد import مكسورة أو ملفات وهمية.
- يجب أن يكون الكود نظيفًا، مفصولًا، قابلًا للصيانة، وقابلًا للتوسع.
- يجب ألا تعتمد العناصر الحرجة على شبكة خارجية أثناء التشغيل.
- يجب أن تكون كل الروابط والمخرجات النهائية قابلة للمراجعة الفعلية.

- اعتماد النسخة النهائية
